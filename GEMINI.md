# Clinical Tracker Project Rules

## Tech Stack
- **Frontend:** Pure ES6+ JavaScript, HTML5, Vanilla CSS.
- **Styling:** Tailwind CSS (via CDN or CLI).
- **Storage:** Browser `localStorage` for offline-first capability.
- **Environment:** Windows PowerShell.

## Design Requirements
- **Responsive:** Support for Desktop, iPad, and Mobile users.
- **Offline:** Must be fully functional without an active internet connection (after initial load).

## Conventions
- Use modern JavaScript features (modules, arrow functions, destructuring).
- Maintain a clean, professional clinical aesthetic using Tailwind.
